# PRODUCTO

Contenido del producto frontend solicitado para la entrega academica.

- `Codigo fuente/`: aplicacion Expo, configuracion, dependencias y cliente API.
- `Scripts SQL/`: no aplica directamente al frontend; los scripts de base de datos se mantienen en el backend.
- `Datos de prueba/`: pendiente de incorporar datos finales de prueba si el equipo los define para la entrega.
- `Librerias y configuracion/`: la configuracion principal esta en `Codigo fuente/package.json`, `Codigo fuente/package-lock.json`, `Codigo fuente/app.json` y archivos de Expo/NativeWind.
